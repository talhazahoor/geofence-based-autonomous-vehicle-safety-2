# Geofence-Based Autonomous Vehicle Safety

## Description
This is a web application designed to enhance autonomous vehicle safety and compliance with regulatory requirements.

## Prerequisites
* Node.js
* PostgreSQL
* Docker

## Installation
1. Clone the repository
2. Run `npm install` to install dependencies
3. Create a `.env` file and add environment variables

## Environment setup
1. Create a PostgreSQL database
2. Update `DATABASE_URL` in `.env` file

## Run commands
* Development: `npm run dev`
* Production: `npm run build` and `npm start`
